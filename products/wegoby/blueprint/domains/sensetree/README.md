# Sensetree Domain

Wegoby's empathic AI—trained on deep user personas to approximate inner experience.

## Overview

Sensetree is the "feeling" side of Wegoby. It models users across multiple dimensions to provide truly personalized experiences.

| Aspect | Description |
|--------|-------------|
| **What** | AI model/service that "experiences" like a user |
| **Foundation** | Emotional Intelligence (EI) research |
| **Approach** | "Becoming the user, not observing the user" |
| **Output** | Experiential preferences guiding system behavior |
